About this documentation site

This site demonstrates common visual HTML elements and a few CSS properties. Each documentation page provides:

  • a short description of the element/property
  • an example code block (visible using <pre>)
  • a rendered Result area so you can see what the code does
  • a link to the official specification (W3C / WHATWG)

Use the navigation above to jump to any page. The pages are structured consistently so you can reuse each page as a template for further documentation.

How to use this site:

  1. Open a page for an element or property.
  2. Read the short description and study the sample code inside the <pre>.
  3. Look at the Result area to see the rendered effect.

CSS notes

Styles in style.css use multiple units (px, em, %), at least two distinct fonts (system sans + Georgia), and color from rgb().
