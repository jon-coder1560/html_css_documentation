The margin Property

The margin property sets the outer spacing of an element. It accepts one to four values and multiple units (px, em, %).

Example Code:

<div class="demo-margin">This box uses margin: 16px (px), padding: 1.2em (em), width: 80% (%) to demonstrate units.</div>

The Result:

This box uses margin: 16px (px), padding: 1.2em (em), width: 80% (%) to demonstrate units.
W3C: margin

Tip

When changing layout on different breakpoints, adjust margin using media queries to preserve flow and spacing.
