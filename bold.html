The b Element

The b element represents a span of text to which attention is being drawn for utilitarian purposes without conveying any extra importance. It applies a stylistic offset—typically bold—without implying additional importance or emphasis.

Example Code:

This is some <b>really bold text</b>.

Note: The characters < and > are shown via character entity references.

The Result:

This is some really bold text.

Spec: W3C: <b> element

Notes & tips

Use <strong> if the text has strong importance (semantic). Use <b> when you need a visual offset only.
