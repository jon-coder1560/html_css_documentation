The i Element

The i element represents a span of text in an alternate voice or mood—commonly used for technical terms, foreign words, book titles, or thoughts. It typically renders text in italics.

Example Code:

Here is an <i>italicized phrase</i> inside a sentence.

The Result:

Here is an italicized phrase inside a sentence.

W3C: <i>

When to use

Use <i> for alternate voice or technical terms; use <em> for emphasis (semantic).
