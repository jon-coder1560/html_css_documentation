The u Element

The u element represents text with an unarticulated, non-textual annotation (e.g., proper names in Chinese). Visually it is typically shown with an underline.

Example Code:

This is an <u>underlined phrase</u> shown for annotation.

The Result:

This is an underlined phrase shown for annotation.

W3C: <u>

Accessibility

Underlines are often used for links; use carefully so users don’t confuse annotated text with navigation.
