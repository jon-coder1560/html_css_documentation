The mark Element

The mark element represents text highlighted for reference or notation purposes (e.g., search matches). Browsers often render it with a yellow background.

Example Code:

Please find the <mark>highlighted word</mark> in this sentence.

The Result:

Please find the highlighted word in this sentence.

W3C: <mark>

Design tip

Keep highlights sparing — too many highlighted sections reduce their usefulness.
