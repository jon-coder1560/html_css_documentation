The color Property

The color CSS property sets the color of text content. It accepts named colors, hex, rgb(), rgba(), hsl(), and more.

Example Code:

<!-- Using rgb() as required by the assignment -->
<p class="demo-red">This text uses rgb(220, 38, 38) for color.</p>

The Result:

This text uses rgb(220, 38, 38) for color.

W3C: color

Design tip

Prefer sufficient contrast for readability. Use contrast checkers when picking foreground/background color pairs.
