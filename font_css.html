The font-family Property

The font-family property specifies prioritized lists of font family names and/or generic family names. Browsers use the first available font in the list.

Example Code:

<!-- fallback list: Roboto → Segoe UI → Arial → sans-serif -->
<p class="demo-slab">This paragraph uses a serif font from the font-family stack.</p>

The Result:

This paragraph uses a serif font from the font-family stack.

W3C: font-family

Notes

Always include generic family as the last fallback: serif, sans-serif, etc.
