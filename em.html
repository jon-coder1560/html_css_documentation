The em Element

The em element marks text that has stress emphasis. Its meaning is semantic: nested <em> tags increase emphasis level.

Example Code:

This is an <em>emphasized</em> word in a sentence.

The Result:

This is an emphasized word in a sentence.

W3C: <em>

Note on semantics

Use <em> when meaning/intonation matters, not just for visual styling (use CSS for style-only changes).
